Slides and Demonstration apps from 2016 PyconIE talk

"Practical Django Debugging and Profiling"

https://python.ie/pycon-2016/schedule/

cprofileexample - Some short scripts for demonstrating cprofile
debugmodeexample - A trivial django app to show debug mode
debugtoolbarexample - A django app to demonstrate the debug toolbar and some extra panels
djangoprofileexample - A django app that generates primes and leaks memory, purely to demonstrate pyvmmonitor
locustexample - A very simple locust file for loadtesting
webpdbexample - A django app for demonstrating webpdb

Every directory has a README with instructions for installing/running
